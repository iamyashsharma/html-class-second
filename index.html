Let's start with hyperlinks, it's command is ("a")


Explaination:- Here 'a' refers to anchor tag.

It's used to make links in a link which is called as hyperlink.It's command is ("a href= 'type here the name of your website'")

when you will click on it then it will move you to that address of website whose url you wrote in href command.

By default it's colour is light blue but you can change it by "color" command.

B/W opening and closing of anchor tag you can write anything which will appear on your display.


Now i will show you example of anchor tags:- using "a href" command for google.com when you will click on it it will move you to google.com
